Microhelp Output

The microhelp output can create various files using the content created in  Microcontent. The output is for developers to use.

Creating a Microhelp Output

Click the + icon in the Output Presets panel to create a new output and configure it as below.

After the preset is configured, use the icons above the list of presets or hover the mouse over a preset in the list to generate the output, publish it, make further edits or duplicate the preset.

The configuration options are described in detail in the online help.

Preset Configuration

General

Format

The format options are HTML5 + CSS3 to create a styled output, plus HTML5, JSON, XML and TXT.

Language

Language sets the select language as metadata for the microhelp file.

Post Generation Script

You can select a script to run after the document has been generated.

Content

Select the required groups to be generated and apply any Condition Expressions to be applied and/or any Variable Sets to be used.
