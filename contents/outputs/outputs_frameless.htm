Frameless Output

Going forward there will be more development of this type of skin. For that reason as well as those below, it is recommended over responsive skins. Any context sensitive help using URLs will need a small change. See Calling Context Sensitive Help.

Frameless outputs are truly frameless, no iFrames at all.

  • iFrames limited how Google indexing and search worked for online help, the frameless skins give improvements there.
  • Each topic has a unique simpler URL
  • The skin editor allows significantly more customization.
  • You can choose to have a Home Page, No Home Page (goes to the default topic) or a Home Page with the default topic as well.
  • A Mini TOC is built in.
  • Advanced users will be able to modify these skins outside of the skin editor or create their own skins as they could in Classic versions. Such editing and creation is done outside of RoboHelp using other HTML editors such as Adobe Dreamweaver. Skins edited or created this way are not supported for further editing in the skin editor.
  • Frameless skins are GDPR compliant. Users have the option to accept or reject the writing of the cookie that records acceptance.
