Responsive HTML5 Output

 Going forward there will be more development of skins for Frameless Outputs so they are the recommended choice. Responsive skins will still be supported.

Responsive HTML5 outputs have proved very popular as with little input from the author, the layout adapts to the user's screen, be that a computer, a tablet or a smartphone. All the author has to do is customize the layout if required.
