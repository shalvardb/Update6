About Adobe RoboHelp

The purpose of the About RoboHelp sample project is to highlight the features, explain how to use them in an easy to follow way and let you see some of them in use.
It is less detailed than Adobe's own online help but the aim is to get you going quickly. For many users it will be sufficient to get started, leaving the online help for the more detailed issues.
Use the project to test things without risking damage to your production projects.

 

This version of the About RoboHelp sample project is synced with Adobe RoboHelp Version 2022.4.

What's In Adobe RoboHelp?

The new and changed features in Adobe RoboHelp are highlighted in What's in Adobe RoboHelp 2022 and explained in the topics in this project.

Users upgrading from Adobe RoboHelp 2019 New UI direct to Adobe RoboHelp 2022 should see Adobe RoboHelp 2020 Changes.

Users upgrading from a Classic version, you should also see Upgrading from Classic Versions.

Upgrading New UI Versions

Upgrading refers to new versions and should not be confused with in-version updates.

If you are upgrading to Adobe RoboHelp 2022 from Adobe RoboHelp 2019 New UI or Adobe RoboHelp 2020, note the Import Settings options on the File menu. These allow you to import your settings into Adobe RoboHelp 2022.

The Sample Projects

The sample projects can be accessed from the Starter Screen when you open Adobe RoboHelp. With Adobe RoboHelp already open, you can access the screen again from the View Menu.

Making Changes

Should you make changes to the sample project that you cannot reverse, you can easily get a fresh copy. Delete the broken copy that is in Documents > My RoboHelp Projects and grab a fresh copy from Program Files. Don't attempt to open it from there, it will not function properly.

C:\Program Files\Adobe\Adobe RoboHelp 2022\resources\data\template\projects\About RoboHelp.

 With each update, both copies of the sample project are updated.
The copy in Program Files is replaced entirely.
In the copy in My RoboHelp Projects the files that are installed are also replaced but any files you have added will be left. If you want to retain your current copy with its installed files, rename the folder in My RoboHelp Project before you apply the update or move it to another location.
